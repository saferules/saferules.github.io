注意 do-while(0) 中可疑的 continue 语句

ID_do_suspiciousContinue
Home › next › previous
continue 语句和 break 语句在语义上是不同的，但在 do-while(0) 中的功效是一样的。 在 do-while(0) 的循环体中如果既有 break 语句又有 continue 语句，continue 语句被误用的可能性较大。 示例：
int foo() { do { .... if (cond1) { break; } .... if (cond2) { continue; // Rather suspicious } .... } while (0); }
建议在 do-while(0) 中只使用 break 语句，不使用 continue 语句。
