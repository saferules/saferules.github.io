拷贝赋值运算符的参数应为同类对象的 const 左值引用

ID_nonStdCopyAssignmentParam
Home › next › previous

拷贝赋值运算符应专注于复制参数的数据，且参数不应按值传递，否则会产生不必要的复制开销以及“对象切片”等问题。

示例：

struct A { A& operator = (A); // Non-compliant };

应改为：

struct A { A& operator = (const A&); // Compliant };
