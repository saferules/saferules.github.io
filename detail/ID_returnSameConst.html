被返回的表达式不应为相同的常量

8.31 ID_returnSameConst
目录 › next › previous

被返回的表达式均为相同的常量是没有意义的。

示例：

bool foo(int a) {
    if (a > 100) {
        return true;
    }
    if (a > 50) {
        return true;
    }
    return true;   // Non-compliant, all the values returned are the same
}
