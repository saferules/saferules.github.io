注意赋值运算符与一元运算符的空格方式

10.4.4 ID_stickyAssignmentOperator
目录 › next › previous

如果 = 与 +、-、*、!、&、~ 等一元运算符之间没有空格，而一元运算符与其子表达式之间有空格，是一种非常怪异的格式，也可能是 +=、-=、*=、&= 等复合赋值运算符的笔误。

示例：

a =+ b;    // Non-compliant
a =- b;    // Non-compliant
a =~ b;    // Non-compliant
a =! b;    // Non-compliant

a += b;    // Compliant
a = -b;    // Compliant
a = ~b;    // Compliant
a = !b;    // Compliant
