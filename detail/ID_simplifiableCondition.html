化简可被合并的逻辑子表达式

ID_simplifiableCondition
Home › next › previous
逻辑或、逻辑与的子表达式可以合并成一个表达式时应尽量合并。 示例：
a < b || a > b // None-compliant, use a != b instead a > b || a == b // None-compliant, use a >= b instead a < b || a == b // None-compliant, use a <= b instead a <= b && a >= b // None-compliant, use a == b instead
不合理的繁琐写法也可能意味着某种错误，需认真对待。
