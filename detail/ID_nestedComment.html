注释不可嵌套

ID_nestedComment
Home › next › previous
嵌套的 /*...*/ 注释不符合标准，/* 与 */ 之间不应出现 /*，某些编译器可以接受嵌套，但不具备可移植性。 示例：
/* // #1 /* // #2, Non-compliant nested comments */ // #3 */ // #4, Non-compliant
根据标准，#1 处的 /* 与 #3 处的 */ 匹配，而 #4 处的 */ 处于失配状态。

依据

ISO/IEC 9899:1999 6.4.9(1) ISO/IEC 9899:2011 6.4.9(1)

参考

MISRA C 2004 2.3 MISRA C 2012 3.1 MISRA C++ 2008 2-7-1
