注释不可嵌套

3.5.1 ID_nestedComment
目录 › next › previous

嵌套的 /*...*/ 注释不符合标准，/* 与 */ 之间不可出现 /*。

示例：

/*                         // #1
    /*                     // #2, non-compliant
    nested comments
     */                    // #3
*/                         // #4, non-compliant

例中 #1 处的 /* 与 #3 处的 */ 匹配，而 #4 处的 */ 处于失配状态。

依据

ISO/IEC 9899:1999 6.4.9(1) ISO/IEC 9899:2011 6.4.9(1) ISO/IEC 14882:2003 2.7(1) ISO/IEC 14882:2011 2.8(1)
