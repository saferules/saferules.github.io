if...else-if 分枝数量应在规定范围之内

ID_if_tooManyElseIf
Home › next › previous
if...else-if 分枝过多不利于阅读和维护，而且执行时需逐一判断各分枝的条件，效率较低，建议改为遵循某种算法的索引结构。 示例：
if (rabbit) { } else if (hamster) { } // ... 3000 branches ... // Computers have the courage to execute, // but do you have the courage to read? else { }
建议 if...else-if 分枝数量不超过 5 个。

配置

maxElseIfCount: 分枝数量上限，超过则报出
