字符串常量中不应存在拼写错误

11.7 ID_literal_misspelling
目录 › next › previous

如果含有拼写错误的常量字符串对用户可见，也可以认为是产品的一种 bug，会对用户造成困扰，故应认真对待。

示例：

void showMessage(int err) {
    if (err) {
        cout << "Error\n";
    } else {
        cout << "Successfull\n";  // Non-compliant
    }
}

例中“Successfull”存在拼写错误，应改为“Successful”。
