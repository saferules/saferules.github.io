作为条件的逻辑表达式不应恒为真或恒为假

ID_invalidCondition
Home › next › previous
以恒为真或恒为假的表达式作为条件是没有意义的，属于逻辑错误。 示例：
void foo() { int i = 1; if (i > 0) { // Non-compliant, always true .... } else { .... // Infeasible path } }
例中变量 i 初始化为 1 后在没有被修改过的情况下，仍对其进行判断是没有意义的，else 分枝的代码得不到执行机会。
