if 分枝和 else 分枝的代码不应完全相同

9.1.4 ID_if_identicalBlock
Home › next › previous

if 分枝和 else 分枝完全相同会使条件判断失去意义，往往是由复制粘贴造成的错误。

示例：

if (condition) { branch } else { branch // Non-compliant }

例中 branch 表示完全相同的代码，需修正本应存在的差异，或去掉 if-else 结构。
