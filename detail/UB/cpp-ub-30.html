被调用函数的语言链接性与该函数定义的语言链接性不符

C++-Undefined-Behavior-30
目录 › next › previous

示例：

// In a.c
int foo() {
    return 0;
}

// In b.cpp
int foo();          // Missing extern "C"

int bar() {
    return foo();   // Undefined behavior
}

依据

ISO/IEC 14882:2003 5.2.2(1)-undefined ISO/IEC 14882:2011 5.2.2(1)-undefined
