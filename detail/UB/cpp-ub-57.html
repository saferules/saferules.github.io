在构造函数或析构函数中调用纯虚函数

C++-Undefined-Behavior-57
目录 › next › previous

示例：

struct T {
    virtual ~T() {
        release();   // Undefined behavior
    }

    virtual void release() = 0;
};

依据

ISO/IEC 14882:2003 10.4(6)-undefined ISO/IEC 14882:2011 10.4(6)-undefined

规则

ID_virtualCallInConstructor ID_virtualCallInDestructor
