用 #define 定义或用 #undef 取消定义具有保留意义的名称

C++-Undefined-Behavior-77
Home › next › previous

如 __LINE__、__FILE__、__DATE__、__TIME__、__STDC__、__cplusplus、defined 等名称被定义或取消定义。

示例：

#undef __cplusplus // Undefined behavior #define __STDC__ 0 // Undefined behavior

依据

ISO/IEC 14882:2003 16.8(3)-undefined ISO/IEC 14882:2011 16.8(4)-undefined

规则

ID_macro_defineReserved ID_macro_undefReserved
