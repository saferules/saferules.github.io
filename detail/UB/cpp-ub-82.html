未经许可，向 posix 命名空间添加声明或定义

C++-Undefined-Behavior-82
目录 › next › previous

示例：

namespace posix {    // Undefined behavior
    ....
}

posix 命名空间是 C++ 的保留命名空间，自定义代码中不应出现名为 posix 的命名空间。

依据

ISO/IEC 14882:2011 17.6.4.2.2(1)-undefined
