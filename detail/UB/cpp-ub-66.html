对模板函数进行非良构调用，或在模板定义和实例化上下文之外有更好的候选函数匹配

C++-Undefined-Behavior-66

示例：

namespace N { struct A {}; template <class T> void foo(T&, double) { .... } } int main() { N::A a; foo(a, 1); } namespace N { template <class T> void foo(T&, int) { .... } // Undefined behavior, a better match for ‘foo(a, 1)’ }
