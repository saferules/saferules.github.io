具有 noreturn 属性的函数不应返回

ID_unsuitableReturn
Home › next › previous
具有 noreturn 属性的函数返回会导致标准未定义的行为。 示例：
[[noreturn]] void foo() { // Use _Noreturn in C if (condition) { abort(); } } // Non-compliant, the function returns if ‘condition’ is false
noreturn 表示不会返回，编译器可能不会为 foo 函数生成跳转回调用处的指令，一旦 foo 函数返回就会造成严重错误。
