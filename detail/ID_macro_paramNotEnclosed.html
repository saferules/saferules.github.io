表达式中的宏参数应该用括号括起来

ID_macro_paramNotEnclosed
Home › next › previous
由于宏只做文本处理，不考虑运算符优先级等问题，故应将宏参数用括号括起来，否则易产生意料之外的错误。 示例：
#define SUM(a, b) (a + b) // Non-compliant
应改为：
#define SUM(a, b) ((a) + (b)) // Compliant

依据

ISO/IEC 9899:1999 6.10.3.5(4) ISO/IEC 9899:2011 6.10.3.5(4)
