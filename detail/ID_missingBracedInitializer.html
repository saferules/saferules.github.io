在初始化列表中对聚合体也应使用初始化列表

6.6.2 ID_missingBracedInitializer
Home › next › previous

结构体、联合体、类对象、数组等聚合体在初始化列表中也应使用由大括号初始化，否则可读性较差。

示例：

int a[2][3] = {1, 2, 3, 4, 5, 6}; // Non-compliant int b[2][3] = {{1, 2, 3}, {4, 5, 6}}; // Compliant struct T {int x, y;}; struct T u[3] = {1, 2, 3, 4, 5, 6}; // Non-compliant struct T v[3] = {{1, 2}, {3, 4}, {5, 6}}; // Compliant
