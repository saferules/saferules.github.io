全局对象不应同时被 static 和 const 等关键字限定

ID_staticAndConst
Home › next › previous
在 C++ 语言中，由 const 或 constexpr 关键字限定的全局对象已具有内部链接性（internal linkage），不应再被 static 关键字限定。 示例：
static const int i = 123; // Non-compliant, redundant ‘static’
应改为：
const int i = 123; // Compliant
