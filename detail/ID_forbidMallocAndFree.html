C++ 代码中禁用 C 内存管理函数

ID_forbidMallocAndFree
Home › next › previous
在 C++ 代码中不应使用 malloc、free 等 C 内存管理函数，应使用 C++ 内存管理方法。 示例：
void foo(size_t n) { int* p = (int*)malloc(n * sizeof(int)); // Unsafe and verbose .... free(p); }
应改为：
void foo(size_t n) { auto p = make_unique<int[]>(n); // Safe and brief .... }
