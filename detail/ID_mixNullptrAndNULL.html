在 C++ 代码中 NULL 和 nullptr 不应混用

17.8 ID_mixNullptrAndNULL
Home › next › previous

在 C++ 代码中 NULL 和 nullptr 不应混用，应统一使用 nullptr。

示例：

void foo(int* a = NULL, int* b = nullptr); // Non-compliant void bar(int* a = nullptr, int* b = nullptr); // Compliant
