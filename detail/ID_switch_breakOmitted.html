switch 语句的每个非空分枝都应该用无条件的 break 或 return 语句终止

ID_switch_breakOmitted
Home › next › previous

在 switch 语句的分枝中，break 或 return 语句的缺失是导致错误的常见原因。

示例：

switch (a) { case 0: b = 1; break; // Compliant case 1: b = 2; // Non-compliant, missing ‘break’ default: b = 3; break; // Compliant }

相连的 case 标签不受本规则约束，但最好配有注释：

switch (c) { case 0: // Compliant case 1: .... break; }

少数情况下，如果确实不能有 break 或 return 语句，应添加注释说明情况，或在 C++ 代码中用 [[fallthrough]] 注明：

switch (v) { case 1: do_something(); [[fallthrough]]; // Compliant, since C++17 default: do_something_default(); break; }
