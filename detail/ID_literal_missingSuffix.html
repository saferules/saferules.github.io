无符号整数常量应具有后缀 U

11.9 ID_literal_missingSuffix
目录 › next › previous

为了便于查看常量的类型，避免意料之外的错误，无符号整数常量应具有后缀 U。

示例（32 位环境）：

0x7fffffff    // Compliant, signed int
0x80000000    // Non-compliant, unsigned int, missing ‘U’
0x80000000U   // Compliant
