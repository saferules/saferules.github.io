有返回值的函数其所有分枝都应显式返回

8.23 ID_notAllBranchReturn
Home › next › previous

如果有返回值的函数在某个分枝没有使用 return 语句显式返回，会导致标准未定义的行为。

示例：

int fun() { if (condition1) { return 1; } else if (condition2) { } // Non-compliant, no return value else { return 3; } }

当条件符合 condition2 时，fun 函数的调用者将得到一个错误的返回值。

例外：

int main() { printf("Hi~\n"); } // Compliant

标准规定未显式返回的 main 函数会返回 0，故 main 函数不受本规则约束。
