不应比较非同类枚举值

ID_differentEnumComparison
Home › next › previous
比较非同类枚举值相当于比较不同类别的事物，没有逻辑意义，往往是设计缺陷或逻辑错误。 示例：
enum Fruit { apple, orange, banana }; enum Pet { hamster, chipmunk, chinchilla }; void foo(Fruit f) { if (f == hamster) { // Non-compliant .... } } void bar(Pet p) { switch (p) { case apple: // Non-compliant .... break; case chipmunk: // Compliant .... break; } }
