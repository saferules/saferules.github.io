内存回收函数不可抛出异常

ID_throwInDelete
Home › next › previous
内存回收函数抛出异常会导致标准未定义的行为。 示例：
class A { .... public: void operator delete(void* p) { if (!p) { throw Exception(); // Non-compliant, undefined behavior } .... } };
