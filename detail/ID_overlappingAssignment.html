不可将对象的值赋给具有部分重叠区域的对象

ID_overlappingAssignment
Home › next › previous
如果两个对象的存储区域有部分重叠，将其中一个对象的值赋给另一个对象会导致标准未定义的行为。 示例：
union U { int16_t x; int16_t y; int64_t z; } u; u.x = u.y; // Compliant u.z = u.x; // Non-compliant
例中 x 和 y 的存储区域完全重叠且类型相同，可以相互赋值；x 和 z 有部分重叠，不可相互赋值。
