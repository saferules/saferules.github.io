用 delete 释放对象不可多写中括号

ID_excessiveDelete
Home › next › previous
用 new 分配的对象应该用 delete 释放，不可用 delete[] 释放，否则导致标准未定义的行为。 示例：
auto* p = new X; // One object .... delete[] p; // Non-compliant, use ‘delete p;’ instead
