不应将类型定义和对象声明写在一个语句中

6.4.7 ID_mixedTypeObjDefinition
目录 › next › previous

将类型定义和对象声明写在一个语句中可读性较差，应分开书写。

示例：

struct T {
    ....
} obj, *ptr, fun();   // Non-compliant

应改为：

struct T {
    ....
};

T obj;   // Compliant
T* ptr;   // Compliant
T fun();   // Compliant
