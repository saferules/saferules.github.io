宏定义中的 # 和 ## 运算符不应嵌套使用

# 和 ## 运算符的求值顺序在标准中是未声明的，不应嵌套使用。 示例：
#define M0(a, b) #a ## b // Non-compliant #define M1(a, b, c) a ## b ## c // Non-compliant #define M2(a) #a // Compliant #define M3(a, b) M2(a ## b) // Compliant
