不应重复检查指针是否为空

14.10 ID_repeatedNullCheck
目录 › next › previous

重复的空指针检查是不必要的，使代码显得繁琐，且干扰编译器优化。

示例：

void foo(int* p) {
    if (!p) {
        return;
    }
    if (p) {   // Non-compliant, ‘p’ is not nullptr
        ....
    } else {
        ....   // Unreachable
    }
}
