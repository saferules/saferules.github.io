不可解引用未初始化的指针

ID_wildPtrDeref
Home › next › previous
未初始化的指针具有不确定的值，对其解引用会导致标准未定义的行为，往往会造成严重错误。 示例：
int foo() { int* p; // Uninitialized return *p; // Non-compliant }
例中指针 p 定义后未被初始化，可能指向不可访问的空间，也可能指向已分配的空间，对其读写可能会导致崩溃，也可能会扰乱程序的行为，造成难以排查的错误。
