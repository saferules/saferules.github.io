不可解引用未初始化的指针

14.3 ID_wildPtrDeref
目录 › next › previous

未初始化的指针具有不确定的值，解引用未初始化的指针会导致标准未定义的行为。

示例：

int foo() {
    int* p;      // Uninitialized
    return *p;   // Non-compliant
}

例中未初始化的指针 p 指向不可预期的地址，通过这种指针读写数据可能会导致崩溃，也可能会扰乱程序的行为，造成难以排查的错误。
