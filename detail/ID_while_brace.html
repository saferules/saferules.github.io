while 循环体应该用大括号括起来

ID_while_brace
Home › next › previous
while 循环体应为大括号括起来的复合语句，即使该复合语句只包含一条语句。 示例：
while (condition) // Non-compliant statement; while (condition) // Non-compliant statement1; statement2;
应改为：
while (condition) { // Compliant statement; } while (condition) { // Compliant statement1; statement2; }
