自增、自减表达式不应作为子表达式

10.2.13 ID_incDecAsSubExpression
目录 › next › previous

自增、自减表达式作为子表达式易使人费解，也易产生求值顺序相关的问题。

示例：

a = ++b + c--;   // Non-compliant

应改为：

++b;             // Compliant
a = b + c;
c--;             // Compliant
