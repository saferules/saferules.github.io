禁用柔性数组

ID_forbidFlexibleArray
Home › next › previous
柔性数组（flexible array）一般是指结构体最后不完整定义的数组成员，表示不占用空间的指针，这种数组在 C99 中有所定义，但不在 C++ 标准之中，在 C++ 代码中不应使用。 示例：
struct A { int len; int dat[]; // Non-compliant }; A* cpy(const A* p) { A* a = (A*)malloc( sizeof(A) + p->len * sizeof(int) ); *a = *p; // Error, only p->len is copied return a; }
例中 *a = *p 这种拷贝赋值运算会漏掉数组的内容，而且数组不会计入 sizeof 的结果，易引起意料之外的错误，所以在 C 代码中也不建议使用柔性数组。

依据

ISO/IEC 9899:1999 6.7.2.1(16) ISO/IEC 9899:2011 6.7.2.1(18)
