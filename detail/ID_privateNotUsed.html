不应存在没有被用到的 private 成员

6.12.6 ID_privateNotUsed
目录 › next › previous

没有被用到的 private 成员是没有意义的，应删除或修正引用关系。

示例：

struct A {
    int foo() { return 1; }     // Compliant

private:
    int bar;                    // Non-compliant, unused
    int foo(int) { return 0; }  // Non-compliant, unused
};
