C++ 代码中参数列表如果为空不应声明为“(void)”

6.5.7 ID_superfluousVoid
目录 › next › previous

与 C 语言不同，在 C++ 语言中空括号和“(void)”均表示没有参数，所以应采用更简洁的方式。

示例：

struct A {
    int foo(void);  // Verbose
    int bar();      // OK
};
