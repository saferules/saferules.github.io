main 函数不应被调用、重载或被 inline、static 等关键字限定

8.2 ID_illFormedMain
Home › next › previous

main 函数作为程序的入口是一种特殊的函数，其链接性由实现定义，不应被调用、重载或被任何函数限定符修饰。

示例：

int main(); // The main function does not need to be declared int foo() { return main(); // Non-compliant } int main(const char* p) { // Non-compliant .... } inline int main() { // Non-compliant .... }
