常量成员函数不应返回数据成员的非常量指针或引用

5.1.19 ID_returnNonConstData
目录 › next › previous

如果常量成员函数返回数据成员的非常量指针或引用，既打破了常量限定，又违反了封装理念，属于不良实现方式。

本规则是 ID_qualifierCastedAway 的特化。

示例：

class A
{
    int i;

public:
    int& foo() const {
        return (int&)i;   // Non-compliant
    }
    ....
};
