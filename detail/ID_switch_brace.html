switch 语句应该用大括号括起来

9.5.14 ID_switch_brace
目录 › next › previous

switch 语句应为包含多条语句的复合语句，且用大括号括起来，否则不应选用 switch 语句。

示例：

switch (v)   // Non-compliant
    case 0:
        foo(v);

应改为 if 语句：

if (v == 0) {   // Compliant
    foo(v);
}
