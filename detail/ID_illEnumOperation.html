枚举对象不应参与位运算或算数运算

10.2.5 ID_illEnumOperation
Home › next › previous

枚举类型的底层整数类型由实现定义，应避免枚举对象参与位运算或算数运算，以防止意料之外的错误并提高可移植性。

示例：

enum { FlagA = 0x1234, FlagB = 0xABCD }; bool foo(uint16_t x) { return (x & FlagA) != 0; // Non-compliant }
