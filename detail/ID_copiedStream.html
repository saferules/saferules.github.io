流式资源对象不应被复制

2.18 ID_copiedStream
目录 › next › previous

FILE 等流式对象不应被复制，如果存在多个副本会造成数据不一致的问题。

示例：

FILE f;
FILE* fp = fopen(path, "r");

f = *fp;                      // Non-compliant
memcpy(fp, &f, sizeof(*fp));  // Non-compliant

依据

ISO/IEC 9899:1999 7.19.3(6) ISO/IEC 9899:2011 7.21.3(6)

参考
