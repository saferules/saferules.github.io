条件编译指令控制表达式的值应为 0 或 1

3.4.8 ID_nonBoolPPCondition
目录 › next › previous

如果 #if 和 #elif 指令的控制表达式均为 bool 型表达式，可使逻辑结构更清晰，有利于阅读和维护。

示例：

#define M 123

#if M   // Non-compliant
....
#endif

#if M != 0   // Compliant
....
#endif

#if defined(M)   // Compliant
....
#endif
