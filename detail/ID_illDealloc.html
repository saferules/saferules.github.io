非动态申请的资源不可被释放

2.16 ID_illDealloc
Home › next › previous

释放非动态申请的资源会导致标准未定义的行为。

示例：

void foo(size_t n) { int* p = (int*)alloca(n); .... free(p); // Non-compliant, ‘p’ should not be freed } void bar() { int i; .... free(&i); // Non-compliant, naughty behavior }

释放在栈上分配的空间或者局部对象的地址会造成严重的运行时错误。

依据

ISO/IEC 9899:1999 7.20.3.2(2)-undefined ISO/IEC 9899:2011 7.22.3.3(2)-undefined ISO/IEC 14882:2003 5.3.5(2)-undefined ISO/IEC 14882:2011 5.3.5(2)-undefined
