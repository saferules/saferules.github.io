头文件中不应定义匿名命名空间

ID_anonymousNamespaceInHeader
Home › next › previous
在头文件中定义匿名命名空间相当于在头文件中定义静态数据，头文件被多个源文件包含时会造成数据冗余。 可参见 ID_staticInHeader 的进一步讨论。 示例：
// In a header file namespace { // Non-compliant void foo(); }

相关
