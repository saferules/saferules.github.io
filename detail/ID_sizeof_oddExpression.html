sizeof 不应作用于比较或逻辑表达式

10.7.2 ID_sizeof_oddExpression
目录 › next › previous

sizeof 作用于 <、>、<=、>=、==、!=、&&、|| 等表达式为常见笔误，相关运算符往往应该移出 sizeof 表达式。

示例：

if (sizeof(a == x)) {  // Non-compliant, may be sizeof(a) == x
    ....
}
if (sizeof(a >= x)) {  // Non-compliant, may be sizeof(a) >= x
    ....
}
