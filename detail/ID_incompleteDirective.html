条件编译代码块应在同一文件中

ID_incompleteDirective
Home › next › previous

#if、#ifdef 与对应的 #else、#elif、#endif 应在同一文件中，否则会增加代码的维护成本。

示例：

// a.h #ifdef M // Non-compliant .... // b.h #else // Non-compliant .... // c.h #include "a.h" #include "b.h" #endif // Non-compliant

示例代码将 #ifdef、#else、#endif 分成了三个文件，使这些文件的依赖关系变得复杂，也使单个文件失去了可读性。
