不应存在紧邻 default 标签的空 case 标签

ID_switch_uselessFallThrough
Home › next › previous
紧邻 default 标签的空 case 标签是没有意义的，应当去除。 示例：
switch (v) { case 0: // Compliant .... break; case 1: // Non-compliant default: case 2: // Non-compliant .... break; }
应改为：
switch (v) { case 0: // Compliant .... break; default: // Compliant .... break; }
